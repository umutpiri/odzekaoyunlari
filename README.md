# odzekaoyunlari
A Quiz game written in React Native. IOS have not tested.
1. Clone the repo
2. Run `npm install`
3. Run `npm start` to test on Android phone.

Google Play Store Page: https://play.google.com/store/apps/details?id=com.odzekaoyunlari&hl=tr
